# aqueduct

```
$ aqueduct --help
usage: aqueduct [-h] [--addacct] [--addreg] [--addtag] [--bootstrap]
                [--delacct] [--delreg] [--deltag] [--ssosetup]

aqueduct v0.0.1

optional arguments:
  -h, --help   show this help message and exit
  --addacct    Add Account
  --addreg     Add Region
  --addtag     Add Tag
  --bootstrap  Bootstrap CDK
  --delacct    Delete Account
  --delreg     Delete Region
  --deltag     Delete Tag
  --ssosetup   SSO Setup
```
